# Multibox required environment variables



| Env variable name       | Meaning                                                                                                                                       |
| ----------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| `AIDBOX_LICENSE`        | Multibox instance license                                                                                                                     |
| `AIDBOX_CLUSTER_SECRET` | Secret to access cluster                                                                                                                      |
| `AIDBOX_CLUSTER_DOMAIN` | <p>Used to route users between the Aidbox instances</p><p>For example: <code>aidbox.example.com</code></p>                                    |
| `AIDBOX_SUPERUSER`      | <p>Root user login and password separated by colon character (<code>:</code>) for Multibox</p><p>For example: <code>login:password</code></p> |
